# Hue_tryout

*Exploring Hue*

---
## Usage

1. **Building**
```
$ make
```
2. **Config** : Modify `config.example.json` and rename to `config.json` \
   [How to get the Hue Bridge ip address and a username](https://developers.meethue.com/develop/get-started-2/)
4. **Executing**
```
$ ./hue_control
```
